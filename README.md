# AI Contour — Умное Автооконтурирование КТ-снимков на базе ИИ

**AI Contour** — это высокотехнологичное десктопное приложение с графическим интерфейсом пользователя (GUI) на PyQt6, предназначенное для автоматического трехмерного сегментирования критических органов и структур риска (OAR) на компьютерных томограммах (КТ) пациента. Программа интегрирует передовые алгоритмы глубокого обучения **TotalSegmentator** и формирует готовый стандартизированный файл разметки в медицинском формате DICOM RTSTRUCT.

Приложение спроектировано в строгом соответствии с принципами чистой архитектуры (вычислительный движок полностью отделен от интерфейса), современным дизайном в темных тонах (Dark Mode) и максимальным удобством для медицинских физиков и радиотерапевтов.

---

## 🚀 Основные возможности и преимущества

### 1. Архитектурное разделение (Клиент-Серверный задел)
Вся вычислительная логика вынесена в полностью независимый сервисный модуль `contour_engine.py`. Это позволяет легко перенести вычисления на удаленный сервер, оставив GUI-приложение в качестве легковесного клиента.

### 2. Динамические пресеты и кастомизация из файла
Программа считывает анатомические пресеты, переводы органов и их цвета из внешнего конфигурационного файла `presets.json`. 
* Если файл отсутствует, он **автоматически генерируется** с дефолтными медицинскими настройками при первом запуске.
* Пользователь может просто отредактировать этот файл, чтобы добавить новые структуры, изменить переводы или настроить цвета.

### 3. Интерактивная кастомизация цветов (ROI Color Picker)
* В списке органов рядом с каждым OAR отображается его точный цвет в системе планирования (в виде цветного квадратика-иконки).
* Выделив любой орган в списке, пользователь может кликнуть на кнопку изменения цвета и выбрать индивидуальный оттенок через встроенную палитру `QColorDialog`. Изменения мгновенно применяются к иконкам и сохраняются в `presets.json`.
* Поддерживается быстрый выбор готовых цветовых палитр: **"Классический AI Contour"**, **"Клинический QUANTEC"**, **"Яркий неоновый"**.

### 4. 3D Постобработка масок
* **Remove small blobs (Удаление шумов):** с помощью 3D-маркировки связных компонентов (`scipy.ndimage.label`) алгоритм отфильтровывает мелкие изолированные группы пикселей (шум нейросети), оставляя только основной монолитный объем органа.
* **Smoothing (Сглаживание контуров):** с помощью 3D Гауссова фильтра (`scipy.ndimage.gaussian_filter`) сглаживается бинарная матрица маски перед передачей в `rt-utils`. Это полностью устраняет эффект «ступенчатости» срезов. Степень сглаживания регулируется параметром `sigma` (от 0.5 до 2.0).

### 5. Новый пресет «Брахитерапия» (Brachy)
Специализированный пресет для гинекологического/урологического оконтуривания органов малого таза:
* Включает мочевой пузырь (**Bladder**), тонкую кишку (**Small Bowel**).
* Автоматически дублирует геометрическую маску кишечника (**Colon**) под двумя уникальными именами: **`Colon`** и **`Colon Dup`** (с немного отличающимся цветом для TPS) для специфики планирования брахитерапии.

### 6. Управление вычислительными ресурсами и точностью
* **CPU / GPU переключатель:** программа автоматически определяет наличие CUDA-совместимой видеокарты (через `torch.cuda.is_available()`). Если она доступна, по умолчанию активен GPU-расчет (сегментация за 15-20 секунд). При ее отсутствии выбор GPU автоматически блокируется с выводом понятного статуса.
* **Три режима точности:**
  * **Стандартная (normal):** разрешение 1.5 мм для высокой детализации контуров.
  * **Быстрая (fast):** разрешение 3.0 мм (флаг `--fast`) для повышения скорости расчета в 3-4 раза.
  * **Ультра-быстрая (faster):** быстрый поиск контура тела целиком (параметр `--task body`).

### 7. Надежность и Индикация
* **Классический информативный прогресс-бар:** показывает точные проценты выполнения по 5-шаговому пайплайну (от конвертации DICOM до записи RTSTRUCT).
* **Анимированный спиннер Брайля:** динамически вращающийся Unicode-значок активности во время расчетов нейросетью предотвращает ощущение "зависания".
* **Цветовая пульсация (Color Pulse Glow):** цвет статуса шага плавно переливается от глубокого синего к неоновому голубому с помощью синусоидального QTimer, создавая ощущение живой работы ИИ.
* **Кнопка мгновенной отмены расчетов ИИ ❌:** во время выполнения сегментации кнопка запуска превращается в яркую красную кнопку `ОТМЕНИТЬ РАСЧЕТ ❌` (остальные поля ввода при этом блокируются). Клик на нее жестко завершает фоновый процесс TotalSegmentator на Windows, выполняет очистку диска от временных файлов и проигрывает предупреждающий звуковой сигнал.

### 8. Интеграция с Elekta Monaco 5.51 и анатомическая группировка
* **Совместимость с Monaco 5.51**: Все названия выходных структур автоматически форматируются согласно требованиям Elekta Monaco. Так, `"lens_left"` конвертируется в `"Lens L"`, `"urinary_bladder"` — в `"Bladder"`, а все остальные парные органы приобретают лаконичные суффиксы `" L"` и `" R"`. Это предотвращает рассинхронизацию имен и некорректную интерпретацию контуров в планирующей системе.
* **6 строгих анатомических блоков**: В интерфейсе и файле пресетов `presets.json` все поддерживаемые 414+ органов структурированы по 6 красивым анатомическим категориям: *Голова и Шея*, *Отделы головного мозга*, *Грудная клетка*, *Брюшная полость*, *Малый таз* и динамическая группа *Остальное* для скелета и мелких мышц.
* **Оптимизированный запуск суб-моделей**: Программа умеет распознавать и автоматически комбинировать несколько специфических суб-моделей ИИ (`brain_structures`, `head_glands_cavities`) с основной моделью `total`. Задачи выполняются последовательно, с аккуратным отключением флага `--fast` там, где это критично для точности мелких структур.

### 9. Улучшенное управление файлами структур (RTSTRUCT)
* **Безопасное слияние контуров**: Полностью исключен небезопасный режим полной перезаписи файлов, предотвращая случайную потерю результатов ручного оконтуривания врача.
* **Выбор целевого файла**: Добавлен удобный выпадающий список выбора конкретного файла RTSTRUCT для слияния (если в папке исследования присутствует несколько файлов).
* **Двунаправленная синхронизация**: Переключение файлов RTSTRUCT во вьюере для просмотра автоматически синхронизируется с выпадающим списком сохранения в настройках ИИ (и наоборот), создавая целостный и интуитивный рабочий процесс.

## 🛠️ Установка и Быстрый Старт

### 1. Создание и активация виртуального окружения
Откройте терминал (например, Windows PowerShell) в корневой папке проекта:
```powershell
# Создание venv
python -m venv venv

# Активация виртуального окружения
.\venv\Scripts\Activate.ps1
```

### 2. Установка зависимостей
Установите все необходимые библиотеки с помощью пакетного менеджера `pip`:
```powershell
pip install -r requirements.txt
```
*Примечание: Для работы ИИ с ускорением на GPU рекомендуется установить PyTorch с поддержкой CUDA, соответствующей версии вашей видеокарты NVIDIA.*

### 3. Быстрый запуск (Windows Batch)
Для быстрого запуска приложения без необходимости вручную активировать виртуальное окружение используйте батник в корневой папке:
```powershell
.\run_app.bat
```
(Запускает приложение в тихом фоновом режиме через `pythonw.exe`, скрывая черное окно консоли).

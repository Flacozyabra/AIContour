# AI Contour — Умное Автооконтурирование КТ-снимков на базе ИИ

**AI Contour** — это высокотехнологичное десктопное приложение с графическим интерфейсом пользователя (GUI), предназначенное для автоматического трехмерного сегментирования критических органов и структур риска (OAR) на компьютерных томограммах (КТ) пациента. Программа интегрирует передовые алгоритмы глубокого обучения **TotalSegmentator** и формирует готовый стандартизированный файл разметки в медицинском формате DICOM RTSTRUCT.

Приложение спроектировано в строгом соответствии с принципами чистого кода, современным дизайном в темных тонах (Dark Mode) и максимальным удобством для медицинских физиков и радиотерапевтов.

---

## 🚀 Основные возможности и преимущества

### 1. Поддержка 41 критического органа (OAR)
Все органы структурированы по анатомическим областям в соответствии с международными протоколами и номенклатурой разметки (**QUANTEC** и **TG-263**):
* **Голова и шея:** головной мозг, спинной мозг, щитовидная железа, череп, трахея, пищевод, левая/правая общие сонные артерии.
* **Грудная клетка:** сердце, левое/правое легкие, аорта, легочная артерия, верхняя полая вена, грудина, левая/правая ключицы.
* **Брюшная полость:** селезенка, левая/правая почки, желчный пузырь, печень, желудок, нижняя полая вена, поджелудочная железа, двенадцатиперстная кишка, левый/правый надпочечники, воротная и селезеночная вены.
* **Малый таз:** мочевой пузырь, предстательная железа, прямая кишка, ободочная кишка, тонкий кишечник, головки левой/правой бедренных костей, кости таза, крестец, левая/правая подвздошные артерии.

### 2. Интеллектуальные режимы работы с RTSTRUCT
* **Режим создания нового файла (New):** генерация нового чистого файла разметки с автозаполнением всех метаданных пациента.
* **Режим слияния (Merge):** автоматическое внедрение новых сгенерированных ИИ контуров в уже существующий файл RTSTRUCT пациента (например, экспортированный из TPS-системы), бережно сохраняя всю исходную ручную разметку и цвета.

### 3. Премиальный UX и Живая Индикация Активности
* **Классический информативный прогресс-бар:** показывает точные проценты выполнения по 5-шаговому пайплайну (от конвертации DICOM до записи RTSTRUCT).
* **Анимированный спиннер Брайля:** динамически вращающийся Unicode-значок активности во время расчетов нейросетью предотвращает ощущение "зависания".
* **Цветовая пульсация (Color Pulse Glow):** цвет статуса шага плавно переливается от глубокого синего к неоновому голубому с помощью синусоидального QTimer, создавая ощущение живой работы ИИ.
* **Автоматическое определение GPU/CPU:** при запуске программа автоматически детектирует наличие графического ускорителя Nvidia с поддержкой CUDA и выводит статус в подзаголовке приложения.

### 4. Кнопка мгновенной отмены расчетов ИИ ❌
* **Умная кнопка двойного действия:** во время выполнения сегментации кнопка запуска превращается в яркую красную кнопку **`ОТМЕНИТЬ РАСЧЕТ ❌`** (остальные поля ввода при этом блокируются).
* **Безопасность и разгрузка CPU:** клик на кнопку отмены (после подтверждения в диалоговом окне) мгновенно останавливает вычисления и жестко завершает фоновый процесс `TotalSegmentator.exe` на Windows, снижая загрузку процессора до нуля.
* **Автоматическая уборка:** после отмены программа проигрывает предупреждающий звуковой сигнал, полностью удаляет временную рабочую папку `temp_autocontour_workspace`, высвобождая место на диске, и разблокирует GUI.

### 5. Звуковое оповещение при завершении 🔔
* Встроенная поддержка нативных, асинхронных звуковых аккордов Windows (`winsound`).
* Приятный звуковой сигнал (системный аккорд) при успешном автооконтурировании.
* Предупреждающий звуковой тон в случае ошибок или ручной отмены расчета.
* Возможность быстрого включения/отключения звука через удобный чекбокс в левой панели с мгновенным сохранением состояния.

### 6. Интегрированная Справка и Дисклеймер
* Кнопка **«Справка и дисклеймер 📖»** в главном меню открывает стильное встроенное руководство пользователя.
* Содержит подробное описание этапов пайплайна, анатомических групп, горячих клавиш, а также важный медицинский дисклеймер.

### 7. Надежность и Автосохранение
* **Умная синхронизация органов:** органы, входящие в несколько областей (например, трахея и пищевод), автоматически синхронизируют свои чекбоксы.
* **Сохранение состояния (QSettings):** приложение сохраняет пути к папкам, выбранный пресет, флаг повышенной точности и настройку звукового оповещения между сессиями.
* Во время выполнения вычислений интерфейс надежно блокируется, защищая процесс от случайных изменений параметров на лету.

## 🛠️ Установка и Быстрый Старт

### 1. Создание и активация виртуального окружения
Откройте терминал (например, Windows PowerShell) в корневой папке проекта:
```powershell
# Создание venv
python -m venv venv

# Активация виртуального окружения
.\venv\Scripts\Activate.ps1
```

### 2. Установка зависимостей
Установите все необходимые библиотеки с помощью пакетного менеджера `pip`:
```powershell
pip install -r requirements.txt
```
*Примечание: Для работы ИИ с ускорением на GPU рекомендуется установить PyTorch с поддержкой CUDA, соответствующей версии вашей видеокарты NVIDIA.*

### 3. Запуск приложения
Запустите главный скрипт программы:
```powershell
python auto_contour.py
```

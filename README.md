# Автоматическое оконтурирование органов риска на КТ для лучевой терапии (MVP)

Этот проект представляет собой Python-скрипт (MVP) для автоматической сегментации органов риска (OAR) на КТ-исследованиях и последующего экспорта полученных контуров в стандартный DICOM RTSTRUCT файл, готовый к импорту в системы планирования лучевой терапии (TPS).

## 🚀 Возможности скрипта

1. **Конвертация DICOM ➔ NIfTI:** Сборка разрозненных срезов КТ в единый 3D-объем с сохранением пространственных метаданных (`dicom2nifti`).
2. **ИИ-сегментация на CPU:** Использование библиотеки `TotalSegmentator` в быстром режиме (`fast=True`), адаптированном для стабильной работы на ПК с 16 ГБ ОЗУ и без мощной дискретной видеокарты.
3. **Очистка памяти:** Принудительная выгрузка моделей и освобождение ресурсов (`gc.collect()`) во избежание сбоев из-за нехватки оперативной памяти.
4. **Сборка масок в DICOM RTSTRUCT:** Корректное транспонирование воксельных масок и запись в стандартный файл RTSTRUCT (`rt-utils`) с жесткой привязкой к координатам исходного КТ-исследования.
5. **Система пресетов OAR:** Возможность фильтрации экспортируемых органов (например, только органы брюшной полости или грудной клетки).

## 🛠️ Быстрый старт (Установка и запуск)

### 1. Создание и активация виртуального окружения
Откройте терминал (например, Windows PowerShell) в папке проекта:
```powershell
# Создание venv
python -m venv venv

# Активация venv
.\venv\Scripts\Activate.ps1
```

### 2. Установка зависимостей
Обновите пакетный менеджер и установите все необходимые библиотеки:
```powershell
python -m pip install --upgrade pip
pip install -r requirements.txt
```

### 3. Запуск скрипта
Запустите пайплайн автооконтурирования, передав пути к папке с КТ-снимками (DICOM) и к выходной папке:
```powershell
python auto_contour.py -i "C:\path\to\ct_dicom" -o "C:\path\to\output_dir" -p abdominal_oar
```

**Доступные пресеты (`-p` / `--preset`):**
* `abdominal_oar` — Органы брюшной полости (селезенка, почки, печень, желудок, аорта, мочевой пузырь, сердце).
* `thoracic_oar` — Органы грудной клетки (сердце, легкие, трахея, аорта, пищевод).
* `all` — Экспортировать все найденные ИИ-моделью структуры.
